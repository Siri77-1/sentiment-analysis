# Sentiment Analysis Web Application

## Overview
A Python-based web application that analyzes customer reviews and predicts whether the sentiment is Positive, Negative, or Neutral.

## Features
- User Login & Registration
- Sentiment Prediction
- Review History
- CSV Report Generation
- SQLite Database Integration

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- Scikit-learn
- Pandas
- NumPy

## How to Run
1. Clone the repository.
2. Install the required packages:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open the application in your browser.
